# IT Salary Predictor

**ML-сервис для прогнозирования зарплат в IT-сфере на основе данных о вакансии.**

---

## О проекте

Сервис предсказывает ожидаемую зарплату для IT-вакансии на основе её характеристик: 
- Название должности
- Требуемый опыт
- Формат работы (офис/гибрид/удалённо)
- Город
- Навыки
- Описание вакансии

Проект включает:
- **ML-модель** для прогнозирования зарплаты
- **REST API** (FastAPI) для интеграции
- **Telegram-бот** для удобного взаимодействия
- **Десктопное приложение** (PyQt6)

---

## Технологии

### Backend
- **FastAPI** — REST API
- **PostgreSQL** — база данных пользователей
- **SQLAlchemy** + **Alembic** — ORM и миграции
- **JWT** — аутентификация
- **bcrypt** — хеширование паролей
- **SMTP** — отправка кодов подтверждения

### ML / Data Science
- **Scikit-learn** — модели (LinearRegression, RandomForest, GradientBoosting)
- **Gensim** — Word2Vec для навыков
- **TF-IDF** — векторизация описаний
- **Pandas** — обработка данных

### GUI
- **PyQt6** — десктопное приложение
- **Aiogram** — Telegram-бот
